# 智图流 (Knowledge Flow)

**智图流** 是一款将学习变成“打怪升级”的智能探索平台。只要输入你感兴趣的任何主题，AI 就会为你量身定制一张专属的“知识地图”，带你一步步解锁新知。

## 🌟 核心理念

学习不应该是一本枯燥的说明书，而应该是一场充满未知的冒险。智图流通过将庞杂的知识体系转化为直观的“技能树”，让你清晰地看到自己的学习进度，并在互动问答中巩固记忆。

## ✨ 亮点功能

- **一键生成知识地图**
  无论是“量子力学”、“咖啡冲煮”还是“文艺复兴艺术史”，只需输入主题，系统瞬间为你构建出层层递进的知识图谱。
  
- **像玩游戏一样学习**
  知识点被转化为一个个待解锁的“关卡”。你需要通过完成当前节点的互动测验，才能解锁下一个更高级的知识领域。
  
- **AI 智能互动测验**
  每个知识点都配备了由 AI 实时生成的专属测验。不是死记硬背，而是通过灵活的问答帮助你真正理解核心概念。
  
- **个性化学习总结**
  完成阶段性学习后，AI 会根据你的答题表现生成专属评价与总结，为你加油打气，并指引接下来的探索方向。
  
- **随时回顾你的旅程**
  系统会自动保存你的每一次探索记录。你可以随时回到过去的知识地图，继续未完成的挑战。

## 🎯 适合谁用？

- **充满好奇心的探索者**：对世界充满好奇，想快速了解一个全新领域。
- **终身学习者**：希望建立系统化知识框架，而不是碎片化阅读。
- **学生与备考党**：需要通过测验来检验学习成果，查漏补缺。
- **任何想让学习变得有趣的人**！

## 🚀 如何开始？

1. 打开智图流。
2. 在首页输入你想了解的任何主题（或者从我们推荐的有趣主题中选择一个）。
3. 点击“开始探索”，等待你的专属知识地图生成。
4. 点击地图上亮起的第一个节点，阅读知识点并开始挑战测验。
5. 答对问题，解锁新节点，继续你的知识冒险！
